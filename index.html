Problem Statement

USC Students need a better way to tell when Dining Hall sections close and open. Not just when the door to the building closes, but when the individual sections close.

Affinity Diagram

Our affinity diagram presents organized ideas for improving dining hall access and updates, including app concepts, email alerts, layout suggestions, social media notifications, and QR code features to keep students informed about section openings, closures, and menu changes.

Sketches

These sketches illustrate a meal map concept for a website or app of thedining halls, focusing on key features like food categories, hall hours, and station openings in a user-friendly interface.

Algorithmic Design Lab 2

The code takes a positive number as input and prints a triangle of asterisks that first increases in size up to that number, then decreases back down. It ensures valid input and outputs the pattern line by line.

Algorithmic Design Lab 3

The code takes temperatures for 10 days, stores them in an array, calculates the average temperature, and then prints the days and temperatures that were strictly below the average. It ensures user input is handled clearly and outputs the calculated average and below-average days.

Algorithmic Design Lab 4

The code prompts the user to enter the dimensions and elements of two matrices, checks if they have the same dimensions, and if so, adds them element-wise to produce a resultant matrix. It then displays the original matrices and their sum in matrix form, or an error if the dimensions do not match.
